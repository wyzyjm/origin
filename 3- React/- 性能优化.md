# 性能优化

-   减少 http 请求
-   较少首屏加载内容.
-   React
    -   路由懒加载, 通过 lazy 方法 加载对应组件.
    -   类组件可以通过 `shouldComponentUpdate`来判断是否需要重新渲染
    -   函数式组件:
        -   通过 memo 方法包裹组件,来缓存组件.
        -   组件内部可以通过 `useMemo`和`useCallback`来缓存结果和 函数.
        -   同级别代码使用 `Fragemennt` 包裹,而不是用 div 或其它标签
        -   使用第三方库,按需加载.
