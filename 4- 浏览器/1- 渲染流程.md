## 渲染流程

1. `域名解析`
    - DNS 缓存
2. `建立TCP链接`
    - 三次握手 和 四次挥手
    - http 和 https 区别
3. `发送http请求`
4. `下载 并渲染网页`

    - http 响应阶段: http 状态码, 304 缓存, http 报文

    - a. 浏览器开辟一个 栈内存,同时分配一个主线程来执行代码. 然后维护一个任务队列.
        - 解析 html 标签.
        - 当遇到 `link script img`等标签, 开启新的线程去加载对应的资源.
        - dom 树.
    - b. 生成 render tree
        - dom 树 + css 树
    - c. 通知 GPU,显卡, 来一点一点绘制.
        - 根据生成的 render tree 来计算位置, 也就是回流.
        - 然后 绘制元素样式, 也就是重绘.
        - 最后 GPU 绘制.
